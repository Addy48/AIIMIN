# Product Language

> Caveman pack. Copy tone.

- Brand: AIIMIN — Life OS, not generic habit tracker
- Headline energy: disciplined, concrete, short
- Waitlist hero: **One screen. Every day.**
- CTA: **Reserve my spot**
- Avoid: purple SaaS clichés, emoji spam in product UI, motivational fluff
- Locked accent: burnt orange `#ff6b35`
- Fonts (waitlist): Familjen Grotesk + Figtree + JetBrains Mono
- Account Arc / Life Arc: steers Daily + Weekly Arc (stored as `tagline`)
