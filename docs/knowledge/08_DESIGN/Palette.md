# Palette (LOCKED)

Do not introduce new brand colors without explicit approval.

## Dark mode

| Token | Hex | Use |
|-------|-----|-----|
| Background | `#1a1a1a` | App background |
| Cards | `#2d2d2d` | Surfaces |
| Accent | `#ff6b35` | Burnt orange — primary accent |
| Completion | `#10b981` | Done / success |
| Incomplete | `#6b7280` | Incomplete / muted |

## Light mode

| Token | Hex |
|-------|-----|
| Background | `#f9f9f9` |
| Cards | `#ffffff` |
| Accents | Same as dark (`#ff6b35`, `#10b981`, `#6b7280`) |

## Grid completion colors

- 8/8 → dark green `#10b981`
- 6–7/8 → lighter green
- <6/8 → gray `#6b7280`
