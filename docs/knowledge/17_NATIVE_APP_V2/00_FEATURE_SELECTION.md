# Feature selection logic (Native V2)

> **Owner revision 2026-07-19 (eve):** Journal voice+type first · Keep-style notes · Calendar lite only · Resumes-only career · Drive usable · **Family docs = priority** · drop large-screen ports.

## Rule

Ship native **only if** ≥3 of: one-hand ≤90s · time-sensitive · glanceable · offline-valuable · daily ritual · ≥2×/week.

**Hard drop** if it needs wide canvas, multi-panel compare, dense month grids, kanban boards, or desktop-only admin chrome.

---

## Primary nav (5 tabs) — revised

| Tab | Job | Why |
|-----|-----|-----|
| **Home** | Ritual: score, habit ticks, next 1–3 events, streak | Daily open |
| **Journal** | Voice + type compose; history | User: first-class |
| **Notes** | Keep-style quick notes | User: first-class |
| **Vault** | Family docs + Drive + Resumes | Priority life admin on phone |
| **More** | Goals, Focus, Discipline, Calendar agenda, Money, Habits manage, Account | Overflow without fake density |

**Removed from primary:** Track hub, Tools grid, Career pipeline, Study tab, Write-hub split.

Secondary: Quick Add `+` · global Search · AI assist sheet · Paywall.

---

## Explicit desktop-only (do not port)

- Placements kanban / full career CRM  
- Calendar month/week multi-panel  
- Finance analytics / wealth  
- Lab graphs / Tech Simulator as home  
- Family heavy multi-column admin (use phone vault list instead)  
- Notes as Notion-like long docs (Keep cards on phone; long edit on desktop)
